How to use the Monthly Tracker
A quick guide to getting the most out of your tracker.
Getting started
+
Type a task in any column and press Enter or click + to add it. New tasks appear at the top of the column. Double-click an existing task name to rename it in place.
/
Press / anywhere on the page (when not already typing) to instantly focus the In progress add input — no mouse needed.
Drag
Drag any task to reorder within a column or move it to another column. The move arrow (→) also lets you switch columns without dragging.
Task buttons
✓
Mark as completed. A note toast appears — optionally type what was delivered, and tick Mark as achievement if this task represents significant work beyond your day-to-day. Press Save, or Skip to dismiss. A completion date is stored and shown on the task.
✕
Mark as cancelled. A note toast appears so you can optionally record the reason — type and press Save, or press Skip or Escape to dismiss it. A cancellation date is stored and shown on the task.
→
Move the task to a different column without dragging.
◇ / ◆
Toggle the ongoing label. Ongoing is a visual marker for long-running work — it does not auto-carry tasks. Use the carry banner each month to decide what moves forward.
✎
Open the note editor to add detail, bullets, dates, or a progress percentage.
✕ (delete)
Delete a task permanently. An Undo option appears for 10 seconds.
↩
Restore a completed or cancelled task back to its original column.
●
Click the coloured dot to cycle priority: Low → Medium → High → Low. Red = High, Amber = Medium, Grey = Low.
↕ Priority
Sort the column by priority (High → Medium → Low). The button changes to ✓ Sorted while the sort is active. It resets automatically when you drag, add, or move a task — sorting is advisory, not locked.
Notes and progress

Click ✎ on any task to open the note editor. Inside you can:

  • Start a line with • for a bullet point — press Enter to continue the list, or Enter on an empty bullet to end it.
  • Use ~~text~~ to mark something as struck through. The eye icon (👁) lets you hide struck-through lines in the note view without deleting them.
  • Click 📅 to insert a date from a calendar picker. Use arrow keys to navigate it.
  • Type a number in the Progress field (0–100) to set a percentage. A colour-coded bar appears on the task card. Setting 100% automatically moves the task to Completed.

Press Cmd+Enter (Mac) or Ctrl+Enter (Windows) to save a note without clicking the Save button.

Notes on completed and cancelled tasks are kept for reference but are not included in the manager update.

Month-to-month carry-over

When you move to a new month, a banner appears if there are incomplete tasks from previous months. It shows the total count and a breakdown by column (e.g. 3 in progress, 2 planned). Click Carry incomplete tasks to this month to bring them all forward — an Undo option appears for 10 seconds if you change your mind. Clicking Dismiss also shows an Undo in case you dismissed by accident.

Completed and cancelled tasks always stay in the month they were resolved and are never carried forward.

Manager update

The summary at the bottom of the page builds automatically from your tasks. Click Copy update to copy it to the clipboard with a generated-on timestamp, ready to paste into an email or message. In-progress, planned, and blocked tasks include their notes; completed and cancelled tasks show their dates only.

Search

Use the search box in the top right to find tasks and notes across all months. The result count and query are shown in the modal title. Each result has a Go to month button to navigate directly to that month. Closing the search clears the input ready for the next search.

Saving, export and import

Data saves automatically to your browser's local storage on every change. Use Export JSON to download a backup — you can choose all months, the current month, or specific months. Use Import JSON (paste or upload) to restore data on another device.

Use Export PDF to generate a printable report. Choose from two report types:

  • Full tracker — all tasks grouped by status, with notes and progress. Useful for month-end reviews.
  • Achievements — only tasks you ticked Mark as achievement when completing them. When you mark a task complete, the note toast includes a checkbox to flag it as an achievement — use this for work that goes beyond your day-to-day, such as significant deliveries, new initiatives, or anything worth highlighting in a performance review. Routine tasks you don't tick are excluded automatically.

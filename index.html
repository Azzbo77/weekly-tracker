How to use the Monthly Tracker
A quick guide to getting the most out of your tracker.
Getting Started
+
Type a task in any column and press Enter or click the + button to add it.
Drag
Click and drag any task to reorder within a column or move between columns.
Task Actions
✓
Mark as complete.
✕
Mark as cancelled.
◉ / ◯
Toggle ongoing (carries forward automatically).
✎
Add or edit notes, bullets, dates or progress.
●
Click the coloured dot to set priority (High / Medium / Low).
Notes & Progress
In the note editor you can:
  • Type • at the start of a line for bullet points (Enter creates a new one).
  • Use ~~text~~ for strikethrough.
  • Click the 📅 icon to insert today's date (or pick any date).
  • Type a number in the Progress box (e.g. 75) and it will show as a bar.
Drag & Drop
Drag any task (the whole item will respond to drag):
• Reorder tasks within the same column
• Move tasks between In progress, Planned and Blocked
• Your exact order is saved automatically for each month.
Manager Update
The box at the bottom generates a clean summary you can copy and paste into emails or documents. Task titles appear as main bullets, with any notes nicely indented underneath for easy reading.
Other Features
↔︎
Use the arrow buttons to move between months.
Export PDF / JSON
Export the current month as a nicely formatted PDF or all your data as JSON.
Import JSON
Restore data from a previous export (old weekly data is automatically converted).
☀︎ / ☾
Toggle between light and dark mode.
Tips
• Double-click a task name to rename it quickly.
• Completed tasks at 100% progress move automatically to Completed.
• Ongoing tasks carry forward to the next month automatically.
• Use the search box (top right) to find tasks across all months.
Import JSON
Paste a previously exported JSON file to restore your data. This will overwrite your current data.
